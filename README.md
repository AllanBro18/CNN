# Facial Expression Recognition menggunakan Convolutional Neural Network

Projek ini berisikan script notebook dan hasil training model klasifikasi ekspresi wajah dengan CNN
 
Dataset yang digunakan: https://drive.google.com/drive/folders/1qeYUNVxNm48Lwt_6d6VUPH8Flh31yiHZ?usp=sharing

# Cara menjalankan program demo
```bash
pip install -r requirements.txt
```

```bash
python emotion_detector.py --mode webcam --duration 60
```

- Berjalan selama 60 detik (default: 30)
- Tekan 'q' untuk keluar
